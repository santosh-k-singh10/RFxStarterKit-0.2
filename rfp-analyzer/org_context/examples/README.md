# Example RFPs and Past Projects

This directory contains examples of past RFPs and successful responses that can be referenced during analysis.

## Directory Structure

```
examples/
├── past_rfps/
│   ├── successful/          # Successfully won proposals
│   └── lessons_learned/     # Projects with valuable lessons
└── requirement_patterns/    # Common requirement patterns
```

## Usage

The RFP analyzer can reference these examples to:
- Find similar past requirements
- Learn from successful responses
- Identify common patterns
- Avoid past mistakes

## Privacy Note

⚠️ **Important**: Do not store sensitive client data or proprietary information in this directory. Anonymize all examples before adding them.

## Recommended Content

- Anonymized RFP documents
- Requirement extraction examples
- Response templates that worked well
- Lessons learned documents
- Common requirement patterns by industry
