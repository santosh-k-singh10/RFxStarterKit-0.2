# Organizational Standards

This directory contains your organization's standards and policies that should be considered during RFP analysis.

## Recommended Files

- `coding_standards.md` - Coding conventions and best practices
- `security_policies.md` - Security requirements and policies
- `architecture_guidelines.md` - Architecture patterns and principles
- `compliance_frameworks.md` - Detailed compliance requirements
- `naming_conventions.yaml` - Naming standards for various artifacts

## Usage

These documents are referenced by the RFP analyzer to ensure extracted requirements align with organizational standards. The context manager can load and provide relevant standards to agents during analysis.
